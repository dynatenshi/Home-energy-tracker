# Трекер расхода электроэнергии

Интеллектуальная микросервисная система для мониторинга энергопотребления с поддержкой ИИ-аналитики. 
Проект позволяет не только собирать данные в режиме реального времени, 
но и получать персонализированные советы по экономии ресурсов.

## Возможности
1. Сбор данных через REST API и обработка потоков данных
2. Использование InfluxDB для эффективного хранения и построения графиков нагрузки
3. Интеграция с Ollama через Spring AI для генерации рекомендаций по оптимизации расходов
4. Асинхронный обмен данными между сервисами через Apache Kafka
5. Полный стек разворачивается одной командой через Docker Compose

## Технологии:
Java 17, Spring (Boot, Web, Data JPA, AI), PostgreSQL, Ollama, InfluxDB, Apache Kafka, Docker

## Запуск

Для запуска необходимо ввести команду в терминал:

```
docker compose -v up -d
```

После создания контейнеров можно получать доступ к функциям через API.

## Остановка

Для завершения работы необходимо ввести команду в терминал:

```
docker compose down
```
